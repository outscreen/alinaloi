Менеджмент проектов программного обеспечния

     Целью курса "Менеджмент проектов программного обеспечения" является обучение студентов и слушателей принципам теории управления проектами по разработке программного обеспечения и приобретения практических навыков планирования, контроля и оптимизации процессов разработки программного обеспечения.

     В результате изучения курса студент / слушатель должен усвоить основные понятия управления проектами, выделять, классифицировать проекты и задачи управления проектами, применять системные подходы и методы управления проектами, владеть современными прикладными средствами поддержки управления проектами.

     Задачей курса является изучение методов управления проектами по разработке программного обеспечения.

     Знания и навыки, приобретаемые пользователями, после прохождения курса. Студент / Слушатель после окончания данного курса должен:

знать:

- Системные подходы и методы управления проектами;

- Критические факторы успеха проекта;

- Концепцию управления рисками, методами и средствами их идентификации, а также оценки и минимизации негативных последствий;

- Управление персоналом проекта в системе управления проектами.

- Методики формирвоания затрат по проекту.

уметь:

- Выделять и классифицировать проекты и задачи управления проектами внутри организации.

- Определять роли участников проекта.

- Формировать организационную структуру для управления проектами.

- Использовать существующие стандарты по управлению проектами.

- Осуществлять выбор программного обеспечения для задач управления проектами, уметь пользоваться Microsoft Project.

     Данный курс расчитан не только на начинающего пользователя. Учебник организован структурно, логично и последовательно от простого к более сложному, таким образом, чтобы приступить к изучению программы пользователю умеющему включить компьютер и запустить приложение в операционной среде.
